# Furnshin

Premium custom furniture, glass works, aluminum solutions, and interior manufacturing in Jaipur, India.

**Website:** [furnshin.com](https://www.furnshin.com)

## Tech Stack

- **Framework:** Next.js 16 (App Router) + React 19
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion + GSAP
- **3D:** React Three Fiber + Three.js
- **Backend:** Google Apps Script + Google Sheets
- **Deployment:** Vercel + GitHub

## Local Development
```bash
# Install dependencies
npm install

# Add environment variables
cp .env.example .env.local
# Fill in your values in .env.local

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

See `.env.example` for all required variables.
Contact the project owner for production values.

## Deployment

Deployed automatically via Vercel on push to `main` branch.
